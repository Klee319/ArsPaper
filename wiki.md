## はじめに

ArsPaperは魔法書にスペルを組み合わせて登録し、マナを消費して発動する魔法プラグインです。

### 基本的な流れ

1. **素材を集める** — アメジストの欠片・ラピスラズリなどからソースジェムを作成
2. **儀式コアと台座を作る** — 作業台でクラフト
3. **スペルブックを作る** — 作業台で見習いの魔法書をクラフト
4. **筆記台でグリフを解放** — 素材と経験値レベルを消費してグリフをアンロック
5. **スペルを組む** — スペルブックをスニーク+左クリックでGUIを開き、形態+効果+増強を組み合わせる
6. **発動** — スペルブック/バインドしたアイテムを右クリック

## マナシステム

スペル発動に必要なリソース。画面上部のBossBarに現在値/最大値が表示される。

### 基本値

| 項目 | 初期値 |
|------|--------|
| 最大マナ | 100 |
| 回復速度 | 5/秒 |
| 回復間隔 | 1秒 (20tick) |
| グリフ解放ボーナス | +5/グリフ |

### マナが増える要素

| 要素 | 効果 |
|------|------|
| グリフ解放 | 1つ解放ごとに最大マナ+5 |
| メイジアーマー装備 | 部位ごとに最大マナ+/回復+ (防具ティアによる) |
| スレッド装着 | マナ上昇スレッド1つにつき+20、回復スレッド1つにつき+2/tick |
| エンチャント | マナ加速/マナ上昇エンチャント |
| ソースベリー | 食べると即座にマナ25回復 |

`/ars status` で現在のマナ内訳を確認できる。

## スペルシステム

### スペルの構成

スペルは **形態(Form)** → **効果(Effect)** → **増強(Augment)** の順で組み合わせる。

```
例: 投射 → 破壊 → 増幅 → 範囲(幅)
    弾を飛ばし、着弾地点のブロックを範囲破壊(増幅で硬いブロックも破壊可能)
```

### スペルブック

| ティア | 名前 | スロット数 | 使用可能グリフTier |
|--------|------|-----------|-------------------|
| 1 | 見習いの魔法書 | 3 | Tier 1 |
| 2 | 魔術師の魔術書 | 6 | Tier 2以下 |
| 3 | 大魔導士の魔導書 | 10 | 全て |

### 操作方法

| 操作 | 動作 |
|------|------|
| 右クリック | 選択中スペル発動 |
| スニーク+右クリック | スロット切り替え |
| スニーク+左クリック | スペル編集GUI |

### スペルバインド

スペルブックのスペルを他のアイテム（剣、ツルハシ等）にバインドできる。

- `/ars spell bind <スロット番号>` — メインハンドのスペルブックからオフハンド/ホットバー9番のアイテムにバインド
- `/ars spell unbind` — メインハンドのアイテムからスペルを解除

バインド済みアイテムは右クリックでスペル発動。

## テレポートシステム

### ウェイストーン

ダイヤモンドブロックの上にのみ設置できるテレポート地点。

- **設置**: ダイヤモンドブロックの上に右クリック
- **名前変更**: スニーク+右クリック → チャットで名前入力 (「cancel」でキャンセル)
- **情報表示**: 右クリック
- **レシピ**: 儀式 — ロードストーン(コア) + エンダーパール×4 + ソースジェム×4 + ダイヤ×2 (1,000 Source)

### テレポートコンパス

ウェイストーンにリンクしてテレポートするアイテム。

- **リンク**: ウェイストーンに向かって右クリック → 座標・名前がLoreに記録
- **テレポート**: 空中で右クリック → リンク先にテレポート（ウェイストーン破壊済みの場合はリンク解除）
- **レシピ**: 儀式 — コンパス(コア) + エンダーアイ×2 + ソースジェム×2 + アメジストの欠片×4 (600 Source)
